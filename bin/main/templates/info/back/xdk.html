XDWorld Digital Earth Construction Technology

XDK is an SDK designed to utilize the XDWorld WebGL engine, an open-source 3D GIS engine.

The XDWorld engine allows for the construction of 3D maps using actual measurement-based terrain, imagery, and infrastructure data on web browsers (IE11 and above, Edge, Firefox, Chrome, Safari).

It can also create a global 3D digital twin space based on the globe. Additionally, it can visualize all data that includes locations and addresses.

XDK is an SDK designed to utilize the powerful 3D GIS engine, XDWorld. It is an API developed for use with various programming languages including Java, C, HTML, and C# to enable the use of 3Ddigital Earth. It allows the construction of 3D maps and the addition of layers to mash up various data types. Through map control and analysis APIs, it can be expanded for analysis, simulation, and statistics.

By adhering to international data standards, it enhances data quality, interoperability, and collaboration, ensuring the scalability of technology to be used consistently and integrated across varioussystems.
